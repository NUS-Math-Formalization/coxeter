1 Order-2-Generated Group

Definition 1.1 ✓
#

Lean declarations

  • OrderTwoGen

An order-two generated group is a group \(G\) with a subset \(S \subseteq G\) such that

  1. Every element of \(S\) has order \(2\).

  2. Every element of \(G\) can be expressed as a product of elements in \(S\).

Given an order-two generated system \((G,S)\), the reflection set is defined as
