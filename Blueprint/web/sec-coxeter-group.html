  • OrderTwoGen.ExchangeProp

\((G, S)\) is an order-two generated system. We say the system satisfy the exchange property if given a reduced expression \(w = s_1 s_2 \cdots s_n \in G\) and \(s \in S\), there exists \(1 \leq i {\lt} n\) such that \(s s_1 \cdots s_n = s_1 \cdots \hat{s_i} \cdots s_n\).

Uses

  • OrderTwoGen.DeletionProp

\((G, S)\) is an order-two generated system. We say the system satisfy the deletion property if given an expression \(w = s_1 s_2 \cdots s_n \in G\), there exists \(1 \leq i \leq j {\lt} n\) such that \(w = s_1 \cdots \hat{s_i} \cdots \hat{s_j} \cdots s_n\).
