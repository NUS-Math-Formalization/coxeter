Theorem 6.3 ✓
#

Uses

  • Lemma 6.2

Lean declarations

  • CoxeterMatrix.strong_exchange

Given a Coxeter matrix \(m\), thus giving the system \((G, S)\). Giving an expression \(w = s_1 s_2 \cdots s_n \in G\), and \(t \in T\), if \(\ell (t w) {\lt} \ell (w)\), then there exists \(1 \leq i {\lt} n\) such that \(t w = s_1 \cdots \hat{s_i} \cdots s_n\). The property is called the strong exchange property.
