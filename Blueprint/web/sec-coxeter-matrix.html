1 Coxeter Matrix

Definition 1.1 ✓
#

Lean declarations

  • CoxeterMatrix

A square matrix \(m : \alpha \times \alpha \to \mathbb {N}\) with non-negative integer entries is a Coxeter matrix if it is symmetric, that is, \(m_{a,b} = m_{b,a}\) for any \((a,b)\); the entry \(m_{a,b}\) is \(1\) if and only if \(a=b\).

Definition 1.2 ✓
#

Lean declarations

  • CoxeterMatrix.ToGroup

Given a Coxeter matrix \(m\), we can construct a group \(G\) with generators \(s_i\) and relations \(s_i^2 = 1\) and \((s_i s_j)^{m_{i,j}} = 1\).

#

  • CoxeterMatrix.SimpleRefl

A set of simple reflections is a subset \(S \subseteq G\) such that every element is obtained by the domain of matrix index set \(\alpha \).
